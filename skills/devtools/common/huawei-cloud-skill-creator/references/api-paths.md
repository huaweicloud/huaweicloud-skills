# API Paths — Phase 2 SDK Source Discovery

This file is generated by Phase 2 (技术调研) when API paths are discovered from SDK source code `_http_info` fields.

## Format

Each entry records the real REST API endpoint discovered from SDK client source:

| SDK Method | resource_path | HTTP Method | Notes |
|------------|---------------|-------------|-------|
| (auto-populated by Phase 2) | | | |

## Discovery Method

```bash
# Method 1: Find all _http_info methods
grep "_http_info" <sdk_path>/{service}_client.py

# Method 2: View specific method's API path
grep -A8 "_{method}_http_info" <sdk_path>/{service}_client.py
# Output "resource_path" key is the real REST endpoint
```

## Rules

- API endpoints are **only** sourced from SDK `_http_info.resource_path` or Huawei Cloud API Explorer
- **严禁** 通过命名模式推断 API 路径
- If SDK has a method but no `_http_info`, mark as ⛔ (API not available via SDK)
