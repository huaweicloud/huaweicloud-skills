## 一、是否需要备案

### 核心规则

- 使用中国大陆节点服务器提供互联网信息服务（网站/App），必须在服务器提供商处完成ICP备案，否则会被阻断
- 境外服务器不需要备案，也不能备案
- 备案是备案顶级域名，顶级域名备案成功后二级/三级域名可直接使用，无需单独备案
- 未备案域名在备案审核期间需关闭解析，管局审核通过后方可解析使用
- 域名在哪里注册购买不影响备案，关键看服务器在哪里

### 子类别与标准答案

#### 1. 境外服务器场景

- **新加坡/海外服务器**：境外服务器不需要备案，也不能完成备案
- **APP海外版**：若APP上架国内应用市场且部署大陆内节点服务器则需备案；仅海外使用则不需要

#### 2. 二级/三级域名场景

- **二级域名是否需单独备案**：不需要。备案是备案顶级域名，顶级域名备案成功后二级域名直接使用
- **三级域名改二级域名**：域名解析华为云大陆内节点服务器都需备案（顶级域名已有备案即可）
- **企业/个人仅支持一级域名备案**：子域名不支持单独备案

#### 3. 备案前后的访问问题

- **备案未完成能否部署**：新增备案需等管局审核通过后才能部署访问；接入备案期间可正常使用
- **备案前官网无法访问（死循环）**：未备案域名备案期间需关闭解析，审核通过后才可使用
- **备案后域名必须解析华为云IP**：在华为云备案成功后需将域名解析至华为云IP；备案成功后可将各级域名解析至华为云

#### 4. 域名解析与备案关系

- **域名解析非华为云IP**：在华为云备案，域名需解析华为云IP；使用其他云商IP建议在对应云商备案
- **海外域名跳转国内已备案域名**：境外域名实际也解析了大陆内节点服务器，需要备案
- **备案后增加网站内容**：若涉及前置审批内容需补充前置审批文件；不涉及前置审批可直接更改网站
- **备案后IP变更**：建议提交变更备案更改IP地址，部分管局会定期核查备案信息一致性

#### 5. 小程序备案

- **小程序备案**：华为云只有网站和APP备案，小程序备案需在小程序提供商处完成

## 二、备案流程与材料

### 核心规则

- 进入华为云备案系统填写备案信息，省份不同规则有差异，需根据省份确定所需材料
- 域名实名认证主体须与备案主体保持一致
- 部分省份要求主体负责人必须为法定代表人
- 备案需要提供绑定的弹性公网IP，仅有内网IP不可
- 一台ECS需包月3个月及以上（含多次续费累计）且为"包年/包月"计费模式才能生成备案授权码

### 子类别与标准答案

#### 1. 备案入口与流程

- **备案流程**：进入华为云官网填写备案信息 → 基础信息校验 → 上传材料 → 华为云初审（1-2个工作日）→ 管局审核（1-20个工作日）
- **非经营性互联网信息服务核准**：与华为云ICP备案相同，在华为云备案系统填写即可

#### 2. 主体负责人要求

- **主体负责人是否必须法人**：省份不同规则有差异，部分省份要求必须为法定代表人
- **法人授权他人备案**：可由法定代表人签署授权书，授权本单位员工作为负责人提交备案
- **港澳居民**：提供港澳居民来往内地通行证或港澳居民居住证

#### 3. 域名与材料要求

- **域名实名认证主体不一致**：建议更改域名实名认证信息，使其与备案主体一致
- **域名实名提示未实名**：若已完成实名认证，可忽略该警告
- **电子营业执照**：上海电子营业执照不支持备案，需提供营业执照原件拍照或彩色扫描件
- **通信地址选不了对应区域**：可优先选择就近区域
- **前置审批**：涉及前置审批行业类别及批复单位参考前置审批文档

#### 4. 服务器与IP要求

- **备案服务器必须绑定公网IP**：备案需提供绑定的弹性公网IP，仅内网IP不可
- **备案IP填写**：填写授权码对应服务器的IP地址，若与实际解析地址不一致可手动更改
- **备案用加WAF的IP**：若后续固定使用的IP地址，填写备案订单时可增加
- **首页URL填写**：网站备案订单中首页URL只能填写一级域名，系统会自动配置

#### 5. 备案授权码

- **授权码生成条件**：ECS需包月3个月及以上（含多次续费累计），且为"包年/包月"计费模式
- **授权码用完**：不支持单独购买，需购买新的可备案服务器生成
- **授权码跨账号使用**：可在有服务器的账号下生成授权码授权给其他账号使用

#### 6. IAM权限

- **IAM账号备案权限**：需授权"Beian Administrator"权限

#### 7. 订单限制

- **一次只能提交一个备案订单**：有进行中的订单需等待审核通过后再提交新订单

#### 8. 上传文件

- **上传文件只支持图片格式**

## 三、接入备案

### 核心规则

- 域名在工信部有备案信息，需解析华为云大陆内节点服务器时，应在华为云完成接入备案
- 新增接入不会影响在其他接入商的备案信息，一个网站或App可备案在多家接入商
- 接入备案期间网站需能正常访问，域名在工信部备案信息需正常
- 取消接入仅取消在华为云的备案信息，其他接入商的备案不受影响
- 接入时可直接上传最新的证件信息

### 子类别与标准答案

#### 1. 接入备案条件与流程

- **在其他云商已备案，转华为云**：在华为云做新增接入备案，不查看原接入商信息
- **不知道原备案平台**：可直接在华为云做接入备案申请
- **接入备案审核时间**：华为云初审1-2个工作日，管局审核1-20个工作日不等
- **接入备案期间能否先使用**：接入备案期间可正常解析华为云服务器使用

#### 2. 接入备案与原备案关系

- **取消接入后ICP还能查到**：只要域名在其他接入商仍有备案，取消华为云接入后工信部系统仍可查到；若没有其他接入商则变成空壳网站/APP，会被管局注销
- **取消阿里或华为一侧接入影响另一侧**：只要域名在使用的云商处已完成接入备案，取消一侧不影响另一侧；若取消后没有其他接入商，会变成空壳网站被管局注销
- **取消接入是否影响其他云服务**：若在其他接入商也有备案则不受影响

#### 3. 接入备案特殊场景

- **原备案证件过期**：接入时可直接上传最新证件信息
- **接入备案需网站可访问**：域名在工信部备案信息正常的情况下，接入期间可解析至华为云服务器
- **事业单位.com域名接入**：根据《互联网政务应用安全管理规定》，事业单位域名应以".cn"或".公益"为后缀备案
- **顶级域名在阿里云备案，二级域名在华为云**：建议用一级域名到华为云完成接入备案
- **填写主体与域名在工信部不一致**：需使用域名对应的主体才能接入备案

#### 4. 已备案后新增IP

- **已备案后购买的EIP/WAF/ELB**：填写备案时会填写实际使用的IP，已备案无需单独对IP做备案
- **备案后IP变更**：建议提交变更备案更改IP

- 接入备案流程：https://support.huaweicloud.com/usermanual-icp/zh-cn_topic_0000002581758204.html
- 基础信息验证：https://support.huaweicloud.com/usermanual-icp/zh-cn_topic_0000002127792637.html

## 四、变更备案

### 核心规则

- 备案信息有变更（主体信息、网站信息、IP地址等）需提交变更备案
- 变更备案期间不影响网站正常使用和访问
- 个人主体不支持直接变更为企业，需注销后用企业材料重新备案
- 同一主体跨省份变更可提交变更主体
- 域名过户后需相应变更备案信息，否则可能被排查

### 子类别与标准答案

#### 1. 主体信息变更

- **公司主体变更**：需提交变更备案修改备案信息
- **法人身份证到期/更换**：可提交变更备案修改法人信息
- **营业执照地址变更**：一般仅地址变更不需提供变更证明；若其他资料日期过期需重新填写签字盖章
- **账号实名变更不影响备案**：账号实名认证变更与备案信息无直接关系，备案信息更改需单独提交变更备案
- **个人变企业**：不支持直接变更，需注销后用企业材料重新备案

#### 2. 网站信息变更

- **备案后增加网站内容**：涉及前置审批需补充文件；不涉及可直接更改
- **IP地址变更**：建议提交变更备案更改IP；同一接入商更换IP不强制重新备案，但最好提交变更
- **服务器删掉/更换**：建议变更备案更改IP地址
- **备案服务器与实际使用不一致**：建议提交变更备案将IP更改为实际使用的IP
- **域名过户**：过户期间备案信息不影响，但过户后需变更备案信息

#### 3. 变更期间业务影响

- **变更/迁移期间网站可正常运行**
- **变更主体期间域名可正常访问**

#### 4. 多云商变更

- **已在其他云商变更**：建议等其他云商变更通过后再提交华为云变更

#### 5. 企业门户变更

- **企业门户IP获取**：可在企业门户账号下查看是否有单独IP，若没有则不需提交变更

### 关键文档

- 变更备案流程：https://support.huaweicloud.com/pi-icp/icp_03_0007.html

## 五、注销备案

### 核心规则

- 注销分为"注销互联网信息服务"和"注销主体"两种
- 注销互联网信息服务：删除主体下某一个互联网信息在工信部的备案信息，注销后该服务不可对外提供
- 注销主体：删除整个主体在工信部的备案信息，主体下所有互联网信息均变为未备案
- 注销后如需再次使用需重新申请ICP备案
- 注销需登录对应备案账号操作

### 子类别与标准答案

#### 1. 注销类型选择

- **主体下只有1个互联网信息**：操作注销主体
- **主体下有多个互联网信息，只注销其中一个**：操作注销互联网信息服务
- **域名不再使用**：可提交注销互联网订单申请

#### 2. 注销后重新备案

- **注销后重新备案**：登录备案系统，在我的备案右侧根据提示填写信息，按首次备案流程操作

#### 3. 特殊注销场景

- **域名被攻击/挂违法网站**：尽快注销备案信息；若之前有公安备案也需一起注销
- **账号无法登录注销**：注销备案需登录对应备案账号，无法找回账号则无法处理
- **域名过期被非法使用**：先注销备案信息，境外服务器不受大陆管控无法举报

#### 4. 注销与取消接入的区别

- **取消接入**：仅取消在华为云的备案信息，其他接入商不受影响
- **注销**：删除在工信部的备案信息

### 关键文档

- 注销互联网信息：https://support.huaweicloud.com/pi-icp/icp_03_0003.html

## 六、账号与主体限制

### 核心规则

- **一个华为云账号只能备案一个主体**，需备案多个主体须使用不同账号
- 个人主体最多备案5个互联网信息服务
- 一台ECS或L实例服务器只能备案5个网站或APP
- 备案与账号实名认证主体可以不一致，两者没有联系
- 域名实名认证主体须与备案主体保持一致

### 子类别与标准答案

#### 1. 一账号一主体

- **一个账号能否备案多个主体**：不可以，一个华为云账号只能备案一个主体
- **已有企业备案能否增加个人备案**：不可以，需使用其他华为云账号
- **同一ICP备案是否支持多个账号**：不可以
- **子公司备案**：若当前账号无备案信息且子公司之前在华为云无备案，可在当前账号备案

#### 2. 备案数量限制

- **个人主体上限**：个人主体最多备案5个互联网信息服务，第6个无法提交
- **一台ECS上限**：一台ECS或L实例只能备案5个网站或APP，超出需使用新的服务器
- **企业主体上限**：因省份而异，参考各地区管局备案要求

#### 3. 跨账号备案

- **域名和资源不在同一账号**：可以备案，备案与域名和服务器在哪个账号购买无关
- **服务器在其他主体账号**：可通过备案授权码授权给当前账号备案
- **备案IP跨账号**：备案填写的IP只要是华为云大陆节点公网IP即可，不限制哪个账号购买

#### 4. 域名与主体关系

- **购买的域名可给其他主体备案**：可以，账号实名认证主体与备案主体可不一致
- **域名实名认证主体须与备案主体一致**
- **备案域名只能一个主体**：一个华为云账号只能备案一个主体

- 个人备案怎么变更到企业备案：https://support.huaweicloud.com/icp_faq/icp_05_0829.html

## 七、APP备案

### 核心规则

- APP使用大陆内节点服务器需完成ICP备案
- APP备案需提供公钥、签名MD5值等特征信息，由APP开发技术人员获取
- APP不支持迁移，需注销原APP后用新主体重新备案
- 同一个APP可以用不同主体备案，但不同主体的APP域名需不同
- APP备案主体更换：可先在新主体下备案成功后再注销原主体APP备案，不影响业务

### 子类别与标准答案

#### 1. APP特征信息

- **公钥/签名MD5获取**：由APP开发技术人员获取，参考APP特征信息获取文档
- **鸿蒙平台公钥和签名MD5**：同上，由技术人员参考文档获取

#### 2. APP备案流程

- **APP备案入口**：进入华为云备案系统按提示填写，与网站备案流程相同
- **APP备案公网IP**：填写华为云大陆节点公网IP即可，不限哪个账号购买
- **APP服务在本地**：ICP备案需使用大陆内节点的云服务器，在哪家云服务商的服务器上部署就在哪家做备案。APP仅部署在本地服务器（非云服务器）的情况下，无法在华为云完成备案，需确认是否有华为云大陆节点的ECS资源

#### 3. APP备案变更

- **变更APP包名**：可操作变更备案增加APP包名
- **APP备案主体更换**：先在新主体下备案成功后再注销原主体APP备案，不影响业务运行

#### 4. APP迁移

- **APP不支持迁移**：需注销原APP后用B主体重新备案
- **同一APP不同主体备案**：可以，但不同主体的APP域名需不同

#### 5. APP图标与备注

- **APP图标**：不同主体可以使用相同的APP图标
- **APP备注**：按实际运营情况如实填写即可

## 八、迁移备案（跨主体）

### 核心规则

- 迁移备案是将A主体的域名/网站迁移至B主体名下，不是更换备案的云商
- 迁移条件：①目标主体需有独立华为云账号（A/B主体不能在同一账号下）②域名实名认证需过户到B公司名下 ③A/B主体不能有正在备案中的订单
- 迁移期间域名在工信部有备案信息的情况下不影响使用
- 迁移后网站备案主体变为新公司，无需再重新备案
- 在华为云的备案想使用阿里云服务器，应做接入备案而非迁移备案

### 子类别与标准答案

#### 1. 网站迁移（跨主体）

- **旧主体域名迁到新主体（不同统一社会信用代码）**：可先注销旧主体域名备案再在新主体重新备案，或通过网站迁移方式直接迁移
- **网站迁移条件**：①B主体需有独立华为云账号 ②域名实名认证过户到B公司 ③A/B主体无进行中订单
- **迁移期间业务影响**：迁移期间域名在工信部有备案信息的情况下不影响使用
- **迁移后是否需重新备案**：迁移后备案主体变为新公司，无需再重新备案

#### 2. 同账号内转移备案

- **备案转移到另一账号**：需登录备案账号完成转移操作
- **账号失效无法登录**：无法找回账号则无法处理转移

#### 3. 迁移与接入的区别

- **迁移到阿里云**：迁移备案是跨主体操作，非更换云商；在华为云的备案想用阿里云服务器应做接入备案
- **注销部分网站迁移到别的主体**：需确认迁入主体所在省份是否支持，支持则可迁移

#### 4. APP迁移

- **APP不支持迁移**：需注销原APP后用新主体重新备案
- **APP备案主体更换**：可先在新主体备案成功后再注销原主体APP备案

- 网站迁移（跨主体）：https://support.huaweicloud.com/pi-icp/icp_03_0022.html

## 九、边界判断

> 以下类型问题不属于ICP备案范畴，应引导用户至正确渠道。

### 非备案问题

| 问题类型 | 说明 |
|---------|------|
| 小程序备案 | 华为云无小程序备案，需在小程序提供商处完成 |
| DNS配置/解析问题 | 非备案类问题，咨询DNS服务 |
| 域名绑定资源问题 | 非备案类问题 |
| 城名注册人信息更改 | 非备案类问题，咨询域名服务商 |
| 华为云账号实名认证 | 非备案类问题，咨询账号服务 |
| 域名实名认证 | 需咨询域名服务商 |
| 云速建站续费/产品问题 | 非备案类问题 |
| 法人注册账号数量限制 | 非备案类问题 |
| 境外服务器解析无法访问 | 非备案类问题，境外无需备案 |
| 网站内容迁移（非备案信息） | 非备案问题 |
| 人工服务/拉会沟通 | 非备案问题 |

### 需提工单处理

| 问题类型 | 说明 |
|---------|------|
| 备案系统页面操作异常/提示找不到修改 | 提交工单处理 |
| 线下华为政务云HCSO平台备案 | 提交工单处理 |
| 号码归属地无法识别 | 反馈专员手动添加 |

### 需用户补充信息

| 问题类型 | 说明 |
|---------|------|
| "备案咨询"/"备案"等笼统问题 | 请用户详细描述问题 |
| "备案后静态页面可以关闭" | 请详细描述问题 |
| 域名备案主体变更（信息不完整） | 确认具体需求 |
