# 账号与实名认证FAQ

> 来源：华为云帮助中心, 2025-12-31
> 原始资料：[账号与实名认证FAQ](../../data/raw/markdown_html/support.huaweicloud.com__icp_faq__icp_05_0012.html__0cc61b1f53.md)

## 概述

本页面汇总了华为云备案过程中关于账号使用与实名认证的常见问题。主要涵盖账号权限要求、实名认证信息一致性等关键前提条件，帮助用户在发起备案申请前确认账号状态是否符合要求。

**关键事实**：
- 备案申请人需使用已完成实名认证的华为云账号进行操作 [来源：doc_0008#账号要求]
- 账号的实名认证主体信息通常需与备案主体信息保持一致 [来源：doc_0008#实名认证要求]

## 详细内容

### 账号要求

详细描述了进行备案操作对华为云账号的基本要求。

**要点**：
- 建议使用主账号进行备案操作，若使用子账号需拥有相关权限 [来源：doc_0008#账号权限]
- 账号需处于正常状态，未被封禁或冻结 [来源：doc_0008#账号状态]

### 实名认证与备案关系

解释了实名认证信息与备案信息之间的关联及一致性要求。

**要点**：
- 个人实名认证账号仅可进行个人性质备案，企业实名认证账号需进行企业性质备案 [来源：doc_0008#主体一致性]
- 实名认证信息变更可能影响正在进行的备案流程 [来源：doc_0008#信息变更]
