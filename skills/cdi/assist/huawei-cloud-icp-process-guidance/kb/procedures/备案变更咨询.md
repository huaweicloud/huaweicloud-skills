# 备案变更咨询

> 来源：华为技术有限公司, 2026-05-27
> 原始资料：[变更主体信息和新增网站能否同时进行](../../data/raw/markdown_html/doc_0015.md), [新增接入时能否同步变更信息](../../data/raw/markdown_html/doc_0016.md), [变更备案](../../data/raw/markdown_html/doc_0195.md), [ICP备案用户指南](../../data/raw/markdown_html/doc_0304.md)

## 概述

备案变更咨询主要涉及备案主体信息、网站信息变更的操作流程、限制条件及相关法规要求。依据《非经营性互联网信息服务备案管理办法》，备案信息发生变更时需提前三十日申请变更备案，否则可能面临网站关闭或注销风险。

**关键事实**：
- 备案信息变更需提前三十日申请，否则可能被注销备案 [来源：doc_0195#操作场景] [来源：doc_0304#1.1 变更备案]
- 变更主体信息与新增网站、新增接入操作存在互斥关系，通常不能同时进行 [来源：doc_0015#变更主体信息和新增网站能否同时进行] [来源：doc_0016#新增接入时能否同步变更信息]
- 变更备案分为变更主体、变更互联网信息服务、变更备案（同时变更两者）三种类型 [来源：doc_0195#变更备案类型]

## 详细内容

### 变更备案类型与范围

变更备案根据修改内容的不同分为三种类型，且存在特定的修改限制。

**变更类型**：
- **变更主体**：仅变更主体信息（如主体负责人、联系方式等） [来源：doc_0195#变更备案类型]
- **变更互联网信息服务**：仅变更网站或App信息 [来源：doc_0195#变更备案类型]
- **变更备案**：同时变更主体信息和互联网信息服务 [来源：doc_0195#变更备案类型]

**修改限制**：
- 省份、主办单位证件号码、域名不支持修改 [来源：doc_0195#PC端操作步骤]
- 若修改主体单位名称，需重新进行域名实名认证 [来源：doc_0195#PC端操作步骤]
- 湖南、贵州管局要求主体负责人变更时需提交”变更备案”并提供新授权书 [来源：doc_0195#变更备案类型]

**已备案域名禁止变更主体规则**：
> 来源：华为技术有限公司, 2025-11-19
> 原始资料：[陕西管局要求](../../data/raw/markdown_html/support.huaweicloud.com__prepare-icp__icp_02_0029.html__720be81ba3.md), [四川管局要求](../../data/raw/markdown_html/support.huaweicloud.com__prepare-icp__icp_02_0036.html__243a993433.md)

**核心规则**：已备案成功的网站/域名不支持变更主体（即不能将备案主体从A公司变更为B公司）。

**各省份管局要求**：
- **陕西管局**：不能变更主体，如营业执照变更请提供变更证明；支持网站迁移，不支持变更主体 [来源：icp_02_0029#变更备案]
- **四川管局**：只允许变更，不允许更换主体。变更时涉及到公司名称时，需上传工商更名函 [来源：icp_02_0036#变更备案]
- **其他省份**：大部分省份遵循相同规则，具体需查看各地区管局要求

**违反规则后果**：
- 若强行变更主体，备案将被驳回或注销 [来源：doc_0195#操作场景]
- 依据《非经营性互联网信息服务备案管理办法》，备案信息不真实将关闭网站或App并注销备案 [来源：doc_0195#操作场景]

**正确处理方式**：
- 如需更换主体，应先注销原备案，再以新主体重新备案 [来源：icp_02_0029#变更备案]
- 部分省份支持”互联网信息迁移”功能，可将网站迁移至其他主体 [来源：icp_03_0022]
- 如主体名称变更（非更换主体），需提供工商更名函后提交变更备案 [来源：icp_02_0036#变更备案]

### 操作冲突与顺序约束

在进行变更备案时，需注意与其他备案操作的互斥关系及执行顺序。

**变更与新增网站**：
- 当备案主体信息变更时，不能同时进行新增网站操作 [来源：doc_0015#变更主体信息和新增网站能否同时进行]
- **华为云主体**：需先提交变更备案，待审核完成后再提交新增网站申请 [来源：doc_0015#主体是在华为云备案的]
- **非华为云主体**：需根据各省管局要求决定顺序。若先新增网站，成功后必须提交变更备案，否则存在注销风险 [来源：doc_0015#主体不是在华为云备案的]

**变更与新增接入**：
- 新增接入和变更备案不能同时提交 [来源：doc_0016#新增接入时能否同步变更信息]
- 部分省份允许在新增接入时变更备案信息，具体需查看各地区管局要求 [来源：doc_0016#新增接入时能否同步变更信息]

**域名过期处理**：
- 若备案号下有域名过期不续费，建议通过变更备案删除该域名，仅保留正常使用的域名 [来源：doc_0135#同一个备案号里面有多个域名，其中域名不续费过期后会对备案号有影响吗？]
- 过期域名若被抢注用于违规内容，可能导致取消接入或注销备案号 [来源：doc_0135#同一个备案号里面有多个域名，其中域名不续费过期后会对备案号有影响吗？]

### 变更备案操作流程

变更备案需遵循标准的审核流程，并注意审核期间的访问影响。

**操作步骤**：
1. 登录华为云备案系统 [来源：doc_0195#PC端操作步骤]
2. 选择变更类型并填写/修改信息 [来源：doc_0195#PC端操作步骤]
3. 上传资料及核验 [来源：doc_0195#PC端操作步骤]
4. 平台初审 [来源：doc_0195#PC端操作步骤]
5. 工信部短信核验（需在24小时内完成） [来源：doc_0195#PC端操作步骤] [来源：doc_0304#1.3 迁移备案]
6. 管局审核 [来源：doc_0195#PC端操作步骤]

**审核期间影响**：
- 单域名网站：变更期间域名可继续访问 [来源：doc_0195#操作场景]
- 多域名网站：若通过变更删除指定域名，该域名在管局审核通过后将无法访问 [来源：doc_0195#操作场景]
