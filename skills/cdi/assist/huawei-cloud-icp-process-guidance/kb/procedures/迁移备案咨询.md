# 迁移备案咨询

> 来源：华为技术有限公司, 2025-12-31
> 原始资料：[在第三方服务商处已备案的网站，接入华为云还要备案吗](../../data/raw/markdown_html/doc_0006.md), [新增接入时能否同步变更信息](../../data/raw/markdown_html/doc_0016.md), [迁移备案](../../data/raw/markdown_html/doc_0200.md) 等

## 概述

迁移备案咨询主要涉及两类场景：一是**接入备案**，即已备案的网站从第三方服务商迁移至华为云（变更接入商）；二是**迁移备案**，即已备案的网站需要更换主体（变更主体）。接入备案后备案号不变，而迁移备案则涉及主体变更的复杂流程与限制。

**关键事实**：
- 网站从第三方服务商转接入华为云，因接入商变更，必须申请新增接入备案 [来源：doc_0006#在第三方服务商处已备案的网站，接入华为云还要备案吗]
- 接入备案是指主体和域名均已通过其他服务商成功备案，现需将服务商变更为华为云，备案号不会发生变化 [来源：doc_0165#接入备案后，备案号会有变化吗？]
- 迁移备案适用于已完成备案的网站需更换主体的场景，可在业务不中断的情况下将网站迁移至新主体 [来源：doc_0200#操作场景（PC端）]

## 详细内容

### 接入备案（服务商变更）

接入备案针对已在外部备案的域名或网站，需将服务商变更为华为云的场景。

**操作要求与限制**：
- **IP地址处理**：服务器迁移到华为云后，原第三方服务商备案的IP地址不需要修改，只需在华为云完成接入备案即可 [来源：doc_0136#在第三方服务商购买了服务器做备案，现在想迁移到华为云，在第三方服务商备案的IP地址需要修改吗]。
- **域名解析风险**：若未在华为云备案直接将域名解析到华为云服务器，网站被排查到后会被阻断，建议尽快完成接入备案 [来源：doc_0144#第三方服务商有备案的域名，在没有做接入备案的前提下，可以直接将域名解析到华为云的服务器上吗]。
- **信息变更限制**：新增接入和变更备案不能同时提交，但部分省份允许在接入时变更备案信息，具体需查看管局规定 [来源：doc_0016#新增接入时能否同步变更信息]。

**流程与时效**：
- **审核时间**：接入备案初审需1-2个工作日，管局审核需1-20个工作日，具体以管局为准 [来源：doc_0006#在第三方服务商处已备案的网站，接入华为云还要备案吗]。
- **服务连续性**：新增接入期间，如原备案信息有效，网站可正常使用；接入成功后域名方可解析至华为云IP [来源：doc_0006#在第三方服务商处已备案的网站，接入华为云还要备案吗]。

### 迁移备案（主体变更）

迁移备案针对需要将已备案网站从一个主体（账号）迁移到另一个主体（账号）的场景。

**约束与限制**：
- **账号要求**：迁入的目标主体必须有独立的华为云账号，且不支持迁移到个人性质的备案主体 [来源：doc_0200#约束与限制]。
- **订单状态**：迁出账号和迁入账号下均不能有正在进行中的订单，否则无法进行迁移操作 [来源：doc_0200#约束与限制]。
- **主体影响**：若原备案主体下只有1个网站或全部迁移，迁移后原主体会被注销或成为空壳主体 [来源：doc_0200#约束与限制]。
- **地区支持**：目前支持迁移备案的地区包括河北省、陕西省、广东省等26个省市自治区 [来源：doc_0200#约束与限制]。

**操作步骤**：
- **入口路径**：通过华为云备案系统“我的备案”页面，选择“更多 > 迁移互联网信息服务”进行操作 [来源：doc_0200#操作步骤]。
- **短信核验**：提交申请后需在24小时内完成工信部短信核验，否则申请会被退回 [来源：doc_0200#操作步骤]。
