# 认领备案咨询

> 来源：华为技术有限公司, 2025-12-31
> 原始资料：[认领备案](../../data/raw/markdown_html/doc_0198.md)

## 概述

认领备案是指将历史备案信息迁移至华为云新备案系统的操作流程。主要适用于2018年9月之前在原系统备案的用户，或涉及跨省变更及迁移互联网信息服务的场景。

**关键事实**：
- 2018年9月之前在原华为云备案系统做过备案的用户，如需继续备案，需先认领备案将历史信息迁移至新系统 [来源：doc_0198#操作场景]
- 在其他接入商处进行迁移互联网信息服务或跨省变更操作后，需进行认领备案 [来源：doc_0198#操作场景]

## 详细内容

### 操作场景

用户在以下场景下需要进行认领备案操作：

1.  **历史系统备案迁移**：2018年9月华为云备案平台升级前，在原系统做过备案的用户，如需新增互联网信息服务或变更备案，必须先完成认领 [来源：doc_0198#操作场景]。
2.  **跨接入商迁移或跨省变更**：用户在其他接入商处完成了迁移互联网信息服务或跨省变更操作后，需在华为云进行认领备案 [来源：doc_0198#操作场景]。

### PC端操作步骤

认领备案支持PC端和App端操作，PC端主要流程如下：

**操作流程**：
1.  **登录系统**：使用华为云账号登录华为云备案系统 [来源：doc_0198#PC端操作步骤]。
2.  **验证备案类型**：系统自动验证备案类型 [来源：doc_0198#PC端操作步骤]。
3.  **上传资料并认领**：单击“上传资料并认领备案”，按要求上传相关资料 [来源：doc_0198#PC端操作步骤]。
4.  **提交认领**：勾选同意协议，单击“提交认领” [来源：doc_0198#PC端操作步骤]。

**注意事项**：
- 对于已备案过的域名，不能重复提交备案申请 [来源：doc_0198#PC端操作步骤]。
- 提交成功后，华为云工作人员将在1~2个工作日内进行审核 [来源：doc_0198#PC端操作步骤]。

### App端操作步骤

用户也可通过华为云App进行认领备案操作，流程包括验证备案类型、提交初审等步骤，具体操作逻辑与PC端一致 [来源：doc_0198#操作步骤]。
