# 备案进度查询

> 来源：华为云帮助中心, 2026-05-27
> 原始资料：[App备案预计审核时间以及进度查询](../../data/raw/markdown_html/doc_0120.md) | [备案进度](../../data/raw/markdown_html/doc_0158.md) | [备案状态](../../data/raw/markdown_html/doc_0188.md)

## 概述

备案进度查询是指用户通过华为云备案系统或工信部备案管理系统查看ICP备案或App备案当前审核状态及进度的操作。备案审核流程主要分为华为云初审（1-2个工作日）和管局审核（1-20个工作日）两个阶段，用户可根据订单状态了解当前所处环节。

**关键事实**：
- 华为云初审时间一般为1~2个工作日，初审通过后提交管局审核 [来源：doc_0120#App备案预计审核时间以及进度查询] [来源：doc_0158#问题二：备案审核需要多长时间？]
- 各省管局审核时间一般在1-20个工作日内，具体时间因各地管局而异 [来源：doc_0120#App备案预计审核时间以及进度查询] [来源：doc_0158#问题二：备案审核需要多长时间？]
- 用户可通过华为云备案系统查询所有状态，或通过工信部系统查询已备案成功的记录 [来源：doc_0158#问题一：怎样了解备案进度？]
- **接入商审核为待修改**状态是最常见的需要用户操作的状态，表示华为云备案专员已审核订单，发现提交的备案申请缺失材料或填写的信息不完整 [来源：doc_0188#备案订单状态一览]
- 用户需单击"审核历史"了解缺失的资料，根据初审意见修改备案申请，然后重新提交接入商审核 [来源：doc_0188#备案订单状态一览]
- **待管局审核**状态不支持任何修改操作，用户需等待管局审核结果 [来源：doc_0188#备案订单状态一览]

## 详细内容

### 查询方式

用户可以通过以下两种方式查询备案进度：

1.  **华为云备案系统**：登录系统查看订单状态，实际进度更新时系统会自动刷新。此方式可查询所有状态的备案订单 [来源：doc_0158#问题一：怎样了解备案进度？] [来源：doc_0188#备案状态]。
2.  **工信部备案管理系统**：仅支持查询已备案成功的ICP备案记录 [来源：doc_0158#问题一：怎样了解备案进度？]。

### 审核时长

备案审核包含华为云初审和管局审核两个阶段：

*   **华为云初审**：提交系统信息后，华为云将在1-2个工作日内完成初步审核 [来源：doc_0158#问题二：备案审核需要多长时间？]。
*   **管局审核**：初审通过后提交至管局，各地管局审核时间不同，一般为1-20个工作日。审核成功后用户会收到短信及邮件通知 [来源：doc_0158#问题二：备案审核需要多长时间？]。
*   **审核通过**：管局审核通过后，将发放备案编号并通过备案系统公布备案信息 [来源：doc_0120#App备案预计审核时间以及进度查询]。

### 备案订单状态一览表

备案订单在不同阶段会显示不同的状态，了解状态含义有助于掌握进度。以下是完整的备案订单状态说明：

| 状态 | 含义说明 | 后续操作 |
| :--- | :--- | :--- |
| **待完善备案信息** | 表示备案订单暂未提交初审。 | 请单击”继续备案”尽快完善备案信息，并提交初审 [来源：doc_0188#备案订单状态一览]。 |
| **等待初审** | 表示备案订单已提交至接入商初审，等待华为云备案专员审核。 | 如需修改订单信息，请先单击”撤销备案”，然后再修改备案申请。如需查询订单历史记录，请单击”审核历史” [来源：doc_0188#备案订单状态一览]。 |
| **接入商审核为待修改** | 表示华为云备案专员已审核订单，发现提交的备案申请缺失材料、或填写的信息不完整，不完全满足ICP备案相关要求。 | **请单击”审核历史”了解缺失的资料，根据初审意见修改备案申请，然后重新提交接入商审核** [来源：doc_0188#备案订单状态一览]。 |
| **已修改待审核** | 表示您已根据初审意见完善订单中的信息、资料，并重新提交，等待华为云备案专员审核。 | 用户无需操作，等待备案专员审核即可。如需修改订单信息，请单击”撤销备案” [来源：doc_0188#备案订单状态一览]。 |
| **初审驳回** | 表示华为云备案专员已审核订单，发现提交的备案申请信息不正确（如真实性核验不通过、网站或App内容存在违规），不满足ICP备案相关要求。 | 请单击”审核历史”了解驳回原因，根据初审意见修改备案申请，然后重新提交初审 [来源：doc_0188#备案订单状态一览]。 |
| **待提交管局** | 表示您提交的备案订单已通过初审，等待华为云备案专员提交备案资料至当地管局进行审核。 | 用户无需操作，等待备案专员提交即可。如需修改订单信息，请单击”撤销备案” [来源：doc_0188#备案订单状态一览]。 |
| **待管局审核** | 表示备案初审已通过，并提交至管局审核。 | **注意：此状态的备案订单，不支持撤销和修改** [来源：doc_0188#备案订单状态一览] [来源：doc_0158#问题一：怎样了解备案进度？]。 |
| **管局驳回** | 表示备案信息或提交的备案资料不符合要求，未通过管局审核。 | 请根据管局驳回意见修改备案信息、资料，并重新提交初审。备案专员初审通过后，会再次将备案提交至管局 [来源：doc_0188#备案订单状态一览]。 |
| **已备案** | 表示已备案成功并获取备案号。 | 如需更改服务器IP地址、修改应急电话、更新营业执照地址等，请参见变更备案，修改备案信息 [来源：doc_0188#备案订单状态一览]。 |

**特别说明**：
- **接入商审核为待修改**状态是最常见的需要用户操作的状态，用户需单击”审核历史”查看缺失资料，然后修改并重新提交 [来源：doc_0188#备案订单状态一览]。
- **待管局审核**状态不支持任何修改操作，用户需等待管局审核结果 [来源：doc_0188#备案订单状态一览]。
- 用户可通过”审核历史”查看订单操作记录，通过”查看详情”查看订单详细信息 [来源：doc_0188#备案订单状态一览]。
