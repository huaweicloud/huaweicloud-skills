# 备案前准备咨询

> 来源：华为技术有限公司, 2025-12-31
> 原始资料：[什么是ICP备案](../../data/raw/markdown_html/doc_0193.md), [准备可备案的服务器](../../data/raw/markdown_html/doc_0210.md), [准备可备案的域名](../../data/raw/markdown_html/doc_0209.md) 等

## 概述

备案是中国大陆的一项法规要求，使用中国大陆节点服务器提供互联网信息服务的用户，必须在服务器提供商处提交备案申请，取得ICP备案号后才能开通访问。备案前需完成账号注册、实名认证、服务器与域名准备等前置工作。

**关键事实**：
- 使用中国大陆节点服务器开办网站或App，必须先办理备案，否则会被阻断访问 [来源：doc_0193#不备案的影响]。
- 通过华为云备案，需要先购买华为云中国大陆节点服务器（如ECS需包月3个月及以上） [来源：doc_0210#准备可备案的服务器]。
- 域名需完成实名认证，且认证信息需与备案主体信息保持一致 [来源：doc_0209#备案域名需满足的条件要求]。
- 一个华为云账号只能备案一个主体，但账号实名认证信息可与备案主体信息不一致 [来源：doc_0254#操作步骤]。

## 详细内容

### 1. 备案基本概念与必要性

备案是根据《互联网信息服务管理办法》等相关法规实施的制度。国家对经营性互联网信息服务实行许可制度，对非经营性互联网信息服务实行备案制度。

**核心要点**：
- **法规要求**：使用大陆节点服务器提供互联网信息服务的用户，需依法履行备案手续 [来源：doc_0177#基本概念]。
- **适用范围**：解析至中国大陆节点服务器的网站和App均需备案；服务器在香港、台湾、澳门或海外则不需要备案 [来源：doc_0081#快速阅读版]。
- **接入商原则**：服务器在哪里购买就在哪里备案。已在其他接入商备案的，转用华为云服务器需办理“新增接入” [来源：doc_0081#快速阅读版]。

### 2. 服务器资源准备

备案前必须拥有符合条件的华为云服务器资源。

**资源要求**：
- **基本条件**：必须购买华为云中国大陆节点服务器。单独的弹性公网IP不能用于备案 [来源：doc_0053#什么是“用于备案的资源”]。
- **ECS要求**：ECS弹性云服务器（含Flexus X实例）需包月3个月及以上，计费模式需为“包年/包月” [来源：doc_0210#备案服务器]。
- **备案次数**：一台服务器可备案次数根据资源类型不同而有所区别，如ECS通常可备案5个网站/App [来源：doc_0057#一台服务器是否可以多次办理备案] [来源：doc_0210#备案服务器]。
- **查看IP**：可在华为云控制台“资源 > 服务视图 > 弹性云服务器 ECS”详情页查看弹性公网IP [来源：doc_0048#如何查看ECS的弹性公网IP地址？]。

**备案授权码**：
- 如需使用同一台服务器为多个主体办理备案，可生成备案授权码。每台ECS可生成5个授权码，每个授权码仅可备案一个网站/App [来源：doc_0210#备案授权码]。

#### 备案授权码完整操作步骤

> 来源：华为技术有限公司, 2026-05-19
> 原始资料：[准备可备案的服务器](../../data/raw/markdown_html/support.huaweicloud.com__prepare-icp__icp_02_0003.html__61ac083daf.md)

**备案授权码定义**：
备案授权码是由弹性云服务器（含华为云Flexus云服务器X实例）/华为云Flexus应用服务器（L实例）生成的用于备案的授权凭证，实际指向该服务器的IP地址 [来源：icp_02_0003#什么是备案授权码]。

**使用场景**：
因一个账号只能备案一个主体，若您需要给不同的主体进行备案，可以注册新的账号，新账号下无备案云资源时可以通过有云资源的账号进行授权，生成备案授权码的方式来完成备案 [来源：icp_02_0003#备案授权码使用场景]。

**生成授权码步骤**：
1. 使用华为云账号登录备案系统（console.huaweicloud.com/beian） [来源：icp_02_0003#获取备案授权码]
2. 单击左侧导航栏的"授权码管理"，进入"备案授权码"界面 [来源：icp_02_0003#获取备案授权码]
3. 单击右上角"生成授权码"，在"生成授权码"窗口中选择云服务和IP [来源：icp_02_0003#获取备案授权码]
   - 选择云服务：请选择一台包月3个月及以上的云服务器资源
   - 选择IP：请选择弹性公网IP资源
4. 根据备案需要选择生成授权码的数量 [来源：icp_02_0003#获取备案授权码]
5. 单击"确定"，生成备案授权码 [来源：icp_02_0003#获取备案授权码]

**生成规则限制**：
- 弹性云服务器需包月3个月及以上（包含多次续费后累计时长），可申请备案授权码 [来源：icp_02_0003#备案授权码生成规则]
- 每台弹性云服务器可生成5个备案授权码 [来源：icp_02_0003#备案授权码生成规则]
- 每个备案授权码只可以备案一个网站或App，不能重复使用 [来源：icp_02_0003#备案授权码生成规则]

**（可选）授权备案授权码给他人**：

1. 打开"备案授权码"界面
2. 选择供其他账号使用的备案授权码，单击"操作"列的"批量授权"
3. 输入授权给的华为云账号名，并单击"验证目标账号"
4. 单击"获取验证码"，并将收到的短信验证码输入至"验证码"栏
5. 单击"确认授权"

**（可选）添加备案授权码自动审批账号**：
若A用户拥有若干授权码并想授权给B用户使用，添加B客户为白名单账号，当B客户备案时输入授权码，将自动审批 [来源：icp_02_0003#添加备案授权码自动审批账号]：
1. 使用华为云账号登录备案系统
2. 单击左侧导航栏的"授权码管理"，进入"备案授权码"界面
3. 单击"自动审批设置"
4. 输入"白名单账号"，单击"添加"
5. 勾选"使用通知"方式

**使用备案授权码**：
1. 登录备案系统 [来源：icp_02_0003#使用备案授权码]
2. 根据界面提示，在"产品验证"页面输入：
   - 云服务类型（资源类型）：选择"备案授权码"
   - 备案授权码：输入具体的备案授权码（系统不会自动填充，请复制粘贴或手动输入）

**注意事项**：
- 一个华为云账号只能为一个主体办理备案 [来源：icp_02_0003#使用备案授权码]
- 同一台服务器如备案多个主体，若其中任何一个备案主体出现问题，可能导致IP被封禁、服务器关停 [来源：icp_02_0003#使用备案授权码]
- 如果使用境内的服务器（ECS）和负载均衡服务，可使用其中一个服务器或服务器生成的备案授权码进行产品验证 [来源：icp_02_0003#使用备案授权码]

### 3. 域名与账号准备

#### 域名要求
域名是备案的重要元素，需满足核验规则。

**关键规则**：
- **实名认证**：域名必须完成实名认证，且持有者信息需与备案主体一致 [来源：doc_0209#备案域名需满足的条件要求]。
- **注册者要求**：个人备案域名注册者应为本人；单位备案域名注册者应为单位（含股东）、主要负责人或高管 [来源：doc_0209#备案域名需满足的条件要求]。
- **核验规则**：备案提交后工信部系统将核验域名顶级域名批复情况、注册服务机构资质、有效期、实名认证状态及主体一致性 [来源：doc_0209#备案域名核验规则（以下5项需同时满足）]。

#### 账号与实名认证

> 来源：华为技术有限公司, 2025-12-31
> 原始资料：[备案主体与华为云实名认证信息不一致，可以吗](../../data/raw/markdown_html/support.huaweicloud.com__icp_faq__icp_05_0090.html__a95d6cc7e1.md), [个人注册账号是否可以给公司备案](../../data/raw/markdown_html/support.huaweicloud.com__icp_faq__icp_05_0092.html__8a769580a3.md)

**账号限制**：一个华为云账号只能备案一个主体（公司或个人），如需为多个主体备案需注册新账号 [来源：doc_0254#操作步骤]。

**账号实名认证与备案主体分离的设计原理**：

华为云账号实名认证信息与备案主体信息可以不一致，这是系统设计的灵活性特性 [来源：icp_05_0090]。

**设计目的**：
- 方便用户管理多个主体的备案需求（通过注册不同账号）
- 支持个人账号为公司备案的场景（如员工为公司办理备案）
- 降低企业备案门槛，无需强制使用公司账号实名认证

**分离规则详解**：
- **账号实名认证类型不影响备案主体类型**：账号实名认证为个人，可以提交企业备案；账号实名认证为企业，也可以提交个人备案 [来源：icp_05_0090]
- **个人账号可为公司备案**：在华为云注册的个人账号可以用于公司备案，账号实名认证信息（个人身份）与备案主体信息（公司信息）可以不一致 [来源：icp_05_0092]
- **备案主体信息以备案时填写为准**：备案主体信息以您在备案申请中实际填写的主体名称、证件信息为准，而非账号实名认证信息

**典型使用场景**：
- **场景一**：员工个人账号为公司备案。员工使用自己的个人实名认证账号，填写公司主体信息完成备案。
- **场景二**：公司账号为子公司备案。母公司账号实名认证，但备案主体填写子公司信息（需注册新账号）。
- **场景三**：个人账号为多个网站备案。一个账号只能备案一个主体，需注册多个账号分别备案不同主体。

**注意事项**：
- 虽然账号实名认证与备案主体可以分离，但每个账号只能备案一个主体 [来源：doc_0254#操作步骤]
- 如需为多个主体备案，需注册新的华为云账号，或使用备案授权码功能 [来源：icp_02_0003#备案授权码使用场景]
- 建议根据实际业务需求合理规划账号使用，避免账号管理混乱

### 4. 特殊场景与材料准备

#### App备案准备
App备案除基础材料外，还需准备App图标、特征信息（包名/Bundle ID、公钥、签名MD5值）等。一次备案可包含多个平台 [来源：doc_0255#App备案前准备]。

#### 前置审批
涉及新闻、出版、教育、医疗保健、药品和医疗器械、文化、广播电影电视节目等特殊行业，需在备案前获得相应的前置审批文件 [来源：doc_0251#前置审批] [来源：doc_0259#前置审批示例图]。

#### 服务内容选择
- **企业/机构**：可根据业务需求选择所有服务内容 [来源：doc_0258#网站备案服务内容目录]。
- **个人**：只能选择“博客/个人空间”或“其他”，选择“其他”需在备注栏说明内容 [来源：doc_0258#网站备案服务内容目录]。

#### 管局特殊要求
不同省份管局有特定要求。例如西藏管局要求企业备案主体负责人必须是法定代表人，个人备案必须使用本省身份证 [来源：doc_0246#企业用户] [来源：doc_0246#个人用户]。
