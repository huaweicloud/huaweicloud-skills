# ICP备案知识问答 · CLI安装指南

## 概述

本文档说明华为云ICP备案知识问答技能的安装和配置要求。

**适用说明**：本技能为纯问答式知识检索技能，不涉及任何华为云CLI（hcloud命令）调用操作。

## 安装要求

### 环境要求

- CodeAgentCLI 或兼容的 Skill 执行环境
- LLM 模型支持（用于语义理解和知识检索）

### 前置条件

本技能为自包含包，内置知识库位于 `./kb/` 目录，无需额外安装依赖。

## 触发验证

技能成功激活后，当用户提问ICP备案相关问题时，会自动从内置知识库检索答案。

### 验证方法

可通过以下方式验证技能正常工作：

1. 触发条件：用户提到「华为云ICP备案需要准备什么材料？」
2. 预期响应：返回ICP备案材料清单，包含身份证、手机等基本要求

## 相关说明
