# ICP备案知识问答 · 验收标准

## 功能验收

### 必须通过

| 编号 | 验收项 | 通过标准 |
|---|---|---|
| AC-01 | Skill激活 | 当用户问题匹配触发条件时，Skill正确激活 |
| AC-02 | 知识检索 | 能够正确读取index.json并匹配相关文档 |
| AC-03 | 内容回答 | 对常见ICP备案问题能给出准确、简洁的回答 |
| AC-04 | 来源标注 | 每个关键事实都标明了来源文档路径 |
| AC-05 | 边界拒绝 | 对非华为云备案、其他云平台、无关问题能正确拒绝 |

### 期望通过

| 编号 | 验收项 | 通过标准 |
|---|---|---|
| AC-06 | 响应格式 | 使用规定的Briefing格式（结论先行，要点分列） |
| AC-07 | 术语一致 | 使用"备案授权码"等华为云官方术语 |
| AC-08 | 诚实回答 | 知识库没有的内容老实说明，不臆造 |

## 质量验收

### 代码质量

| 验收项 | 标准 |
|---|---|
| SKILL.md大小 | 单文件 ≤ 500行 |
| 总目录大小 | ≤ 5MB |
| JSON有效性 | index.json符合JSON格式规范 |
| 路径引用 | 所有引用的文档路径真实存在 |

### 文档完整性

| 验收项 | 标准 |
|---|---|
| Frontmatter | name/description/tags齐全，推荐version |
| 正文章节 | 概述→前置条件→核心命令→参数确认→参考文档（必需章节） |
| 参考文档 | verification-method.md存在（推荐） |

## 安全验收

| 验收项 | 标准 |
|---|---|
| 凭证泄露 | 无AK/SK、密码等敏感信息硬编码 |
| 外部链接 | references/目录仅引用本地知识库，不链接外部网站 |
| 输入安全 | 对用户输入进行基本校验，防注入 |

## 体验验收

| 验收项 | 标准 |
|---|---|---|
| 回答长度 | 单次回答控制在500字以内 |
| 拒绝话术 | 礼貌、专业，不伤害用户体验 |
| 歧义处理 | 问题模糊时能主动反问澄清 |

## 测试用例集

### TC-01：材料咨询
- **输入**："华为云ICP备案需要准备什么材料？"
- **预期**：触发并返回个人/单位备案材料清单，标明来源

### TC-02：驳回处理
- **输入**："备案被驳回了怎么处理？"
- **预期**：触发并返回常见驳回原因及处理方法

### TC-03：授权码
- **输入**："授权码怎么生成的？"
- **预期**：触发并返回授权码生成方法和使用限制

### TC-04：跨平台拒绝
- **输入**："阿里云备案流程是什么？"
- **预期**：触发但礼貌拒绝，说明是华为云知识库

### TC-05：无关问题
- **输入**："今天天气怎么样？"
- **预期**：不触发，说明超出ICP备案范围

### TC-06：知识盲区
- **输入**："某个非常冷门的备案问题"
- **预期**：触发但老实说明"暂无收录"

## 验收结论

**通过条件**：AC-01 ~ AC-05 全部通过，AC-06 ~ AC-08 尽量通过
