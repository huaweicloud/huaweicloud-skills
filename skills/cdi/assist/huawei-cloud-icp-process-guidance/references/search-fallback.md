# 搜索兜底策略

当知识库无法满足需求时，通过网络搜索作为补充手段。

## 触发条件

按以下顺序判断是否启用搜索补全：

| 优先级 | 条件 | 操作 |
|-------|------|------|
| P1 | 知识库检索结果为空 | 立即搜索 |
| P2 | 知识库内容时间戳超过6个月 | 搜索最新政策对比 |
| P3 | 用户明确要求查询最新进展 | 搜索补全 |
| — | 其他情况 | 仅使用知识库 |

## 搜索命令

使用 `mcporter` 调用 Exa 搜索（来自 agent-reach）：

```bash
# 华为云ICP备案相关搜索
mcporter call 'exa.web_search_exa(query: "华为云ICP备案 查询 2026", numResults: 3)'
```

### 搜索 Query 构造规则

```
华为云ICP备案 + [核心关键词] + [年份]
```

示例：
- `华为云ICP备案 授权码 2026`
- `华为云ICP备案 驳回处理 2026`
- `华为云网站备案 材料 2026`

## 截断规则

1. 找到明确定义则立即停止搜索
2. 最多检查3个结果
3. 保留最权威来源（华为云官网 > 政务平台 > 第三方）

## 输出融合

当使用搜索补全时，在标准格式末尾追加：

```
**网络补充**（仅供参考，以官方最新公告为准）：
· 来源1：[标题](URL)
· 来源2：[标题](URL)
```

示例：

```
**网络补充**（仅供参考，以官方最新公告为准）：
· 来源1：[华为云ICP备案帮助中心](https://support.huaweicloud.com/icp-filing/)
· 来源2：[工业和信息化部ICP备案管理系统](https://beian.miit.gov.cn/)
```

## 边界控制

### 可以搜索的场景

- 华为云ICP备案最新政策变动
- 具体管局要求差异
- 实操经验分享（社区/论坛）
- 知识库文档发布时间较早时的交叉验证

### 禁止搜索的场景

- 竞品对比（阿里云、腾讯云备案）
- 价格、购买、续订相关
- 非华为云的技术问题
- 涉及个人信息查询

## Disclaimer 要求

所有搜索结果必须附带免责声明：

> **注意**：网络信息可能存在时效性和准确性差异，建议以华为云官方最新公告为准。如有疑问请联系客服 950808 转3。

## 与常驻规则的配合

本策略是「常驻规则」中"知识库无果时的兜底方案"，与以下规则配合使用：

- **规则1**：先体检 — 本 skill 无多后端概念，但需判断是否超出 ICP 备案范围
- **规则2**：声明策略 — 开始补全搜索前说明"知识库暂无覆盖，尝试网络搜索"
- **规则3**：重试链 — 搜索失败时按 agent-reach 的 Exa 重试链处理（见 search.md）
